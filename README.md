# bert-qa-chatbot
BERT-based Question Answering Chatbot using Tkinter and Hugging Face API.
This is a GUI-based Question Answering chatbot built using:
- Python (Tkinter)
- Hugging Face API
- BERT Model (bert-large-uncased)

## Features
- User-friendly GUI
- Context-based question answering
- Uses BERT model for accurate responses
- Displays confidence score

1. Install dependencies:
